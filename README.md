# EliteHunt - 精英狩猎助手

一个为最终幻想14打造的精英狩猎助手插件。

## 功能特点

- 🗺️ **地图视图** - 显示当前地图的狩猎目标位置
- 📊 **S怪计数器** - 追踪S级精英怪物的击杀次数
- ⚙️ **设置界面** - 自定义插件的各种功能选项
- 🔊 **语音通知** - 当发现精英怪物时播放提示音

## 安装方法

1. 下载 `EliteHunt.zip`
2. 解压到 XIVLauncherCN 的插件目录
3. 在游戏中启用插件

## 使用说明

- 打开插件主窗口：`/eh`
- 打开计数器窗口：`/eh counter`
- 重新加载数据：`/eh reload`

## 依赖项

- OmenTools.dll
- System.Speech.dll
- GuerrillaNtp.dll
- TinyPinyin.dll
